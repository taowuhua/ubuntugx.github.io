CSS3 2D Transforms

w3schools
CSS3 transforms allow you to translate, rotate, scale, and skew elements.
CSS3 2D Transforms

  • translate() 移动
  • rotate() 旋转
  • scale() 缩放(width, height)
  • skewX() 沿 x 轴旋转
  • skewY() 沿 y 轴旋转
  • matrix() 将几种旋转整合起来 matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())

CSS Selectors[attribute]

参见 css-trick && w3schools

[data-value] {
  /* Attribute exists */
  /* 包含这个属性 */
}

[data-value="foo"] {
  /* Attribute has this exact value */
  /* 属性有这个确定的值 */
}

[data-value*="foo"] {
  /* Attribute value contains this value somewhere in it */
  /* 属性的值包含这个字符串 */
}

[data-value~="foo"] {
  /* Attribute has this value in a space-separated list somewhere */
  /* 值在一组以空格分隔的属性值之中，或者是单独的一个值 */
}

[data-value^="foo"] {
  /* Attribute value starts with this */
  /* 属性的值由这个字符串开始，区分大小写 */
}

[data-value=|"foo"] {
  /* Attribute value has this in a dash-separated list somewhere */
  /* 匹配破折号前面的字符串或唯一值 value|=1999 匹配 1999-2000 */
}

[data-value$="foo"] {
  /* Attribute value ends with this */
  /* 属性的值由这个字符串结尾，区分大小写 */
