setIntereval() 和 bind()

setInterval() 里的 this 问题

setInterval 的作用: 周期性的调用一段函数或执行一段代码。

var intervalID = window.setInterval(func, delay[, param1, param2, ...]);
var intervalID = window.setInterval(code, delay);
  • intervalID 是这个重复操作的唯一标识符，可以作为参数传给 clearInterval()
  • func 是想要重复调用的函数
  • code 是另一种语法的应用是由一段字符串构成的代码（不推荐，原因见下）
  • delay 是每次执行延迟的毫秒数，函数每次调用会在延迟之后发生，和 setTimeOut 一样，实际延迟的时间可能会稍长一点

基本用法：

var intervalID = setInterval(animate, 1000)

但是 this 在 function 中调用时，因为是普通方式调用，所以在非严格模式下指向 global object 不正确
这时候可以用 bind
